# grupo3_pfe_2026.1-code

- Este repositório é destinado ao projeto de desenvolvimento de software / projeto front-end, onde nesta etapa utilizamos exclusivamente HTML e CSS.

- O foco principal desta fase é a construção da estrutura visual da aplicação, definindo layout, organização dos elementos, semântica das páginas e identidade visual.

- Paralelamente ao desenvolvimento da interface, foi elaborada a documentação completa do projeto, descrevendo objetivos, requisitos, regras de negócio, fluxos da aplicação e planejamento das próximas etapas.

- Essa documentação serve como guia para o desenvolvimento, garantindo que a implementação visual esteja alinhada com o funcionamento esperado do sistema.


## Essa divisão por etapas facilita a organização do projeto, a manutenção do código e a clareza no processo de desenvolvimento.
